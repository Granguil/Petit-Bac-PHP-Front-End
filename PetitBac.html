Règles et Bonnes Pratiques

L'objectif de ce jeu est de trouver un mot commençant par la lettre et correspondant au thème proposé. Si vous ne trouvez pas, vous pouvez passer au suivant.

Attention : l'orthographe est importante.

Cas particulier :

  1. Nom Propre (Ville, ...) : si il y a É ou une autre lettre accentuée en début de mot, mettez une minuscule accentuée.
  2. Personnage Célébre : il y a deux cases à remplir comme suit :
    • Nom et Prénom : C'est simple, en premier le nom puis le prénom dans la seconde case.
    • Nom seul ou pseudo : Remplir uniquement la première case.
    • Nom suivi d'un le, la, d', de, ... : Remplir uniquement la première case, en mettant comme suit l'article en minuscule et le qualificatif commençant par une majuscule : Guillaume le Conquérant.
  3. Evénement Historique : il y a trois cases à remplir :
    1. La première est le type de l'événement si il rentre dans le nom comme 'Bataille' pour Bataille de Marigan mais pour reconquista, il faut mettre aucun.
    2. La seconde sert à indiquer la jonction entre le type et le nom : le 'de' de Bataille de Marignan.
    3. La troisième sert à donner le nom de l'événement.
  4. Oeuvre d'Art : l'article (le pour le Penseur) et le nom de l'oeuvre(livre, cinéma, peinture, série, ...).
